Services Interoperability API #

The API allows any service to store or upload data for a specific user, using a key/value approach.

It is important to note that the data is expected to be encrypted, and HyperID does not have access to the encryption keys. This means that the data can only be accessed by authorized parties who have the appropriate encryption keys.
To set a data:
  1. In the key field, enter key.
  2. In the value field, enter value.
  3. Click Set Key.
