Retrieving on-chain MultiPass NFT  information (on-chain KYC status). The user must claim their MultiPass NFT to their wallet via the User's Portal, which is not available for sandbox demo. So in this example, the information is provided for a test wallet address that has already claimed a MultiPass NFT.
